PrintNow

Web-to-print software
06/05/2011

http://www.printnow.com/

Prologic Inc., now registered as PrintNow Technologies Inc. has  been developing pre-press solutions  since 2003.  Six years ago they started the PrintNow web to print project.  Their web-to print cloud-hosted SaaS is packaged as:

1. PrintNow Small Business –  For  small and simple format printing business requirements. Start up cost is $5,000.

2. PrintNow Business Plus – For small to medium print service providers that handles small and large format printing.  Top the $10,000 start-up cost and the system can be scaled to handle up to 10 storefronts.

3. PrintNow Enterprise – Suits large companies with extensive customization needs.  The package is claimed to be able to support unlimited number of storefronts.  Initial investment is more than $10,000.

All three PrintNow web-to-print packages have these basic components:

  • Storefront –  PrintNow web to print storefront can be a retail storefront with product browsing and online ordering applications, or a corporate storefront for internal print service management  through the User Management , Order Fulfillment,and Reporting Modules.  The .Net Master Pages platform used by PrintNow allows branding and skinning of pages of the storefront.

  • Quote Calculator and Job Submission tools available through the storefront which are used to instantly quote print job and shipping prices, and to upload printjobs, respectively, are 2 features available only to the Enterprise package.

  • Admin Tools -System settings like storefront payment options, shipping, user groups, and merchant accounts, as well as category definition, tags, colors and fonts, price mapping, and SEO can be configured and managed through the Admin Panel.

  • Online Flex Editor –  The Online Editor has an Adobe flex interface that allows interactive personalization and instant proofing. However,  some advanced capabilities of this tool, like  Large Format printing and Color SVG Artwork are not available for the Small Business web to print system, while backgrounds, lay-outs, themes, and cliparts library, and 3D product preview are available only for the the Enterprise package.

4. Merge Engine –  To enable dynamic imaging, compressed delivery and online proofing, PrintNow has the Merge Engine tool  which supports most image file format such as PDF, EPS, PS, AI, SVG, PSD, TIF, JPG, and PNG.

Optional Modules

Shipment and inventory data can be integrated through a standard API that can be provided by PrintNow at no cost,  but if clients require custom system engineering and integration, template production, and website building,  PrintNow offers such services on a per project basis for additional fees. The following are all of the optonal modules available:

  • Additional Storefront
  • Super Sites (Multi-Tenant)
  • API Integration
  • Chat Application
  • Data List Management
  • FTP Delivery
  • Security Certificate (SSL)
