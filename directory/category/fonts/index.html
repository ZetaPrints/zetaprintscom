    Free online font converter

    Fonts
    21/01/2011

    http://onlinefontconverter.com/

    Useful for an occasional free font conversion between Mac and Windows. Comes handy if you are just starting out with web-to-print and are not prepared to invest in a full-featured font editing tool like FontLab.

    A notable feature is font extraction from a PDF file. It may help on larger texts, but a small PDF file like a business card is unlikely to contain all the glyphs.
