Initial Magento config

Web-to-print store builder
15/08/2011

These are the most important settings that you should configure after installing Magento. Navigate to System / Configuration in your Magento admin panel.

Basic configuration for web-to-print and dynamic imaging

There are quite a few settings that can be configured here, we are only interested in the relatively crucial ones to get your web-to-print store up and running. Lets start from the top and work our way down through the tabs.

Configuration Tab Section Role and recommended value
General Web Session cookie management By default Magento cookie lifetime is set at 3600 seconds (1 hour). This means that customers will be asked to log in after 60 minutes of inactivity which can be pretty annoying and cost you a sale.
Recommended value: 86400 (24 hours)
Design Themes Read more about Magento web-to-print themes.
HTML Head
  • Upload a favicon image file, also known as “favorites icon”. Image size should be 16 x 16 px.
  • Default Description and Keywords – Provide description and keywords for SEO purposes.
Header
  • Logo Image Src – URL path to the logo image file on your server
  • Logo Image Alt – Alternative text for the logo image
  • Welcome text – Welcome message displayed in the header before customers log in.
Footer Copyright
Curency setup Currency Options Set a default currency for your store. Default value is US Dollar.
Store email addresses Store email addresses Set General, Sales and Support email addresses that will appear as sender signatures in transactional emails.
Contacts Contact Us Enable or Disable Magento native contact form.
Recommended value: Yes
Catalog ZetaPrints web-to-print Settings Configure web-to-print synchronization settings.
Sales Checkout Checkout options Allow guest checkout if you want to permit customers to place orders without having to register accounts in your store.
Recommended value: No
Shipping settings Settings Set shipping origin physical address. Read more about shipping configuration.
PayPal Merchant account Input email address associated with PayPal merchant account.
Payment methods Saved CC, Check / Moneyorder Enable payment methods that you want accepted in your store.
Advanced Admin Security Admin login session time in seconds.
Recommended value: 3600 (1 hour)
Advanced Disable Modules Output This is where you can disable most of the Magento native modules, including:

  • Newsletter
  • Sitemap
  • Wishlist
  • Tags etc.

 Advanced configuration for web-to-print

Magento has many more tabs and settings to configure. Eventually, you will need to check them all to make your web-to-print store look polished and function efficiently.

We recommend to allocate about an hour of your time and simply go through each and every of them. It will also help you familiarize yourself with what is possible in your brand new web-to-print store.
