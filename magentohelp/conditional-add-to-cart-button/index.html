Conditional “Add to cart” button

Web-to-print for developers
08/02/2011

Our web-to-print extension for Magento is configured to hide the original Add to cart button in order to prevent adding web-to-print products to a customer’s cart before they are personalized. The button is re-enabled right after customers update all previews.

Fake Add to cart button

Absence of original button may cause confusion to customers. Therefore, a fake Add to cart button is added to preserve consistent store interface.

Use following CSS selectors to style fake Add to cart button and text notice under it:

  • #zetaprints-fake-add-to-cart-button – for the button
  • #zetaprints-fake-add-to-cart-warning – for the notice
