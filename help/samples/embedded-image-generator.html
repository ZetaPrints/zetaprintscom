Web-to-print and dynamic image generator embedding.

This page is a static HTML page that acts as a wrapper for an embedded ZetaPrints page.

The embedded page has no menus. Your wrapper is supposed to provide all management features.

Explanations

  • IFRAME src URL: http://realestate.zetaprints.com/?page=template;TemplateID=23EFEE1C-32C9-470F-92BC-1D0ACA25B8F2;RetT=id;RetE=1
  • ?page=template;TemplateID= - required part
  • RetT=id - return orderID upon completion (RetT param is required)
  • RetE=1 - embedded page, drop menus.
  • ; - note that parameters are separated by ;
  • Referral URL - page embedding and return parameters are only valid if a referral URL is present in the inital HTTP request. E.i. running a page on your desktop produces no referral URL, so it won't work.
