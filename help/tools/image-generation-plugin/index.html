Dynamic imaging widget

ZetaPrints dynamic imaging and web-to-print Flash widget can be embedded into any website as easily as a YouTube video. The widget can show templates from an existing catalog of designs or you can upload your own. It’s free and open source.

About

The Flash widget can be used anywhere you need to produce a personalized image using a template:

  • e-cards
  • badges
  • web page elements (buttons, headers, tabs, etc.)
  • email signatures

The widget retrieves templates from ZetaPrints and talks to our dynamic imaging API to process them. You can use any templates from our design marketplace or upload your own. The dynamic imaging widget can be embedded in a page like the one you see here or take a full page.

Embedding code for existing catalogs

All designer catalogs accessed via our design marketplace have Embed button in the top pagination bar. The link opens a small window with a sample widget and the embedding code.

All public catalogs have a sample widget and the embedding code on Catalog Overview page.

Custom embedding

To embed from our server

Insert this HTML code snippet into your web page. Change the following parameters to suit:

  • width / height – in pixels or %
  • catalog’s RSS feed (20 latest designs only) – copy and paste the feed’s address to replace the default one

To embed from your server

Copy the source file from our location: http://www.zetaprints.com/flash/e-cards-plugin.swf

Change the following parameters to suit:

  • location of the source file – change to where you copied e-cards-plugin.swf
  • width / height – in pixels or %
  • catalog’s RSS feed – copy and paste the feed’s address to replace the default one

Applications

We built this dynamic imaging plugin for fun, but there is much more in it than showing funny pictures on blogs.

  • Promotion of your products – build a catalog with simplified samples of your print products and ask other websites to embed the plugin. It will drive more traffic to your website as people click through to see more designs.
  • Promotion of your talent – build a catalog with interesting designs and let people have fun with them and see your name as the author.
  • Integration – embed the plugin into your pages with dynamically generated content or web-to-print integration as we did for Magento.
